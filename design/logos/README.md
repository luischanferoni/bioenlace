# Logos Bioenlace (borradores SVG)

Opciones basadas en **nodos + puente / B** (1), **ondas + punto** (6) y **continuidad / enlace** (7).

## Archivos

| Archivo | Idea |
|--------|------|
| `concept-1a-nodos-puente.svg` | Dos nodos y un arco (puente). |
| `concept-1b-nodos-doble-puente.svg` | Dos arcos entre los mismos nodos (doble vínculo). |
| `concept-1c-nodos-b-monograma.svg` | Letra B con trazos redondos y nodos a la derecha. |
| `concept-6a-ondas-concentricas.svg` | Círculos concéntricos + punto central. |
| `concept-6b-senal-wifi.svg` | Arcos tipo señal + punto emisor. |
| `concept-6c-ondas-y-enlace.svg` | Ondas + mini puente entre dos puntos. |
| `concept-7a-vesica-enlace.svg` | Dos círculos que se cruzan (vesica) + elipse de acento en el centro. |
| `concept-7b-infinito-continuidad.svg` | Símbolo infinito (doble trazo). |
| `concept-7c-cruz-entrelazada.svg` | Dos elipsas gruesas a 90° (encaje). |
| `concept-rombo-rejilla-punta.svg` | Rombo, cruz interior mínima, tramo a la punta en `#ff6b6b` con círculo. |

## Inkscape

- Abrí cada SVG en **Inkscape 1.x**: capas `Fondo` / `Marca`; grupos con `inkscape:label` para el ícono.
- Colores de referencia: `#093e4d`, `#38be7f`, `#ff6b6b`.
- Para **vesica** precisa en `7a`: seleccioná los dos círculos y usá **Intersección** (booleana) en lugar de la elipse central si querés la lente exacta.
- Favicon / app: duplicá el archivo, ajustá el `viewBox` al grupo `Marca` y exportá PNG o WebP.
