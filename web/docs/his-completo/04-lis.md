# Laboratorio (LIS)

**Madurez orientativa:** 2/4 (integración externa, no LIS propio)

## Lo que tenemos

- [x] Traer resultados de laboratorios externos (estándar clínico internacional).
- [x] Guardar informes y analitos para consulta en Bioenlace.
- [x] Paciente y profesional consultan en app y asistente.
- [x] Sincronización programada (cron / lote).
- [x] PDF de informe para el paciente.

## Lo que falta para un LIS hospitalario completo

- [ ] Orden de laboratorio end-to-end dentro del HIS (pedido → muestra → validación en planta).
- [ ] Workflow por sector de laboratorio.
- [ ] Catálogo analítico operativo (perfiles, rangos, LOINC).
- [ ] Interfases estándar con cualquier LIS del mercado más allá de proveedores configurados hoy.
- [ ] Liberación y auditoría de resultados por rol de laboratorio.
